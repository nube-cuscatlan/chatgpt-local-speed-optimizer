# Privacy Policy

Local Speed Optimizer for ChatGPT is a local-only browser extension.

## Data collection

The extension does not collect, sell, transmit, or share personal data.

## Message content

The extension does not copy, store, or transmit the content of ChatGPT conversations. It counts conversation DOM nodes and hides older nodes locally to reduce visual rendering load.

## Storage

The extension uses Chrome storage for:

- User settings, such as whether optimization is enabled and how many recent messages remain visible.
- Local optimization statistics, such as total visible/hidden message counts and estimated memory reduction.

These values remain in the browser storage area and are not sent to any external server by the extension.

## Network requests

The extension does not make external API requests. The donation link is a normal user-clicked link and is not required for the extension to work.

## Affiliation

This is an independent extension and is not affiliated with OpenAI.
